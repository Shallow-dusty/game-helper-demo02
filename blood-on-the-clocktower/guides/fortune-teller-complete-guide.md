## 1. 角色基础信息

### 基本属性

| 属性 | 内容 |
|------|------|
| **英文名称** | Fortune Teller |
| **中文名称** | 占卜师 / 预言家 |
| **阵营** | 好人阵营 (Good Team) |
| **类型** | 镇民 (Townsfolk) |
| **剧本** | Trouble Brewing (TB) |
| **图标** | 🔮 |
| **难度** | ⭐⭐⭐ (中等) |

### 能力描述

**官方规则 (Official Rule):**
> 每晚，你选择两名玩家。说书人会告诉你，他们中是否有恶魔。

**英文原文:**
> Each night, you learn if either of the 2 players you choose is a Demon.

**能力细节:**
- **选人方式**: 你每晚主动选择两名玩家（可以包括自己或死亡玩家）
- **信息类型**: 是/否二元信息（Yes/No）
- **触发时机**: 每晚（包括首夜和其他夜晚）
- **特殊机制**: 红鲱鱼机制（Red Herring）

## 2. 能力机制详解

### 2.1 基础运作流程

**夜间行动步骤:**

1. **唤醒阶段**: 说书人在夜间顺序中唤醒占卜师
   - **首夜**: 在厨师、共情者之后
   - **其他夜**: 在小恶魔、守鸦人之后

2. **选择玩家**: 占卜师指向两名玩家
   - 可以选择任何玩家（包括自己、死亡玩家、旅行者）
   - 不能只选一名玩家，必须选择两名

3. **接收信息**: 说书人给出反馈
   - **点头 (Yes)**: 两人中至少有一人是恶魔或红鲱鱼
   - **摇头 (No)**: 两人都不是恶魔，也不是红鲱鱼

4. **重新入睡**: 占卜师闭眼，说书人继续夜间流程

### 2.2 红鲱鱼机制 (Red Herring)

**什么是红鲱鱼？**

红鲱鱼（Red Herring）是血染钟楼中一个独特的误导机制，专为占卜师设计。

**运作原理:**
- 在游戏设置阶段，说书人会秘密指定一名**好人玩家**作为"红鲱鱼"
- 当占卜师查验包含红鲱鱼的组合时，说书人会给出"是"的反馈
- 这会让占卜师误以为查到了恶魔，但实际上该玩家是好人

**红鲱鱼的特性:**

| 特性 | 说明 |
|------|------|
| **阵营** | 必须是好人阵营（镇民或外来者） |
| **唯一性** | 整局游戏只有一个红鲱鱼 |
| **不可变** | 红鲱鱼不会因角色变化而改变（除非该玩家死亡或变成邪恶阵营） |
| **保密性** | 红鲱鱼本人不知道自己是红鲱鱼 |
| **选择权** | 说书人可以在游戏开始前或首夜选择红鲱鱼 |

**红鲱鱼可能是占卜师自己吗？**

是的！在较小的游戏（5-7人局）中，说书人可能会将占卜师自己设为红鲱鱼。这样：
- 占卜师选择"自己+任何其他玩家"时，总会得到"是"
- 这反而能提供额外信息：如果占卜师连续查到多次"是"，可能是自己是红鲱鱼
- 通过排除法，占卜师可以推断其他玩家的清白

### 2.3 特殊情况处理

#### 情况 1: 查验到隐士 (Recluse)

**隐士能力**: "可能被判定为邪恶/爪牙/恶魔"

- 说书人可以选择让隐士在被占卜师查验时显示为恶魔
- 这会导致占卜师得到"是"的反馈，即使隐士是好人
- 隐士可以与红鲱鱼同时存在，增加混淆

#### 情况 2: 被下毒或醉酒

**中毒 (Poisoned)**: 被投毒者下毒
- 占卜师的能力完全失效
- 说书人会给出错误信息（可以是任意反馈）
- 占卜师本人不知道自己被下毒

**醉酒 (Drunk)**: 酒鬼角色
- 如果占卜师实际是酒鬼（以为自己是占卜师）
- 说书人会给假信息，可能真也可能假
- 用于平衡游戏，防止信息过于准确

#### 情况 3: 查验死亡玩家

- 占卜师可以选择死亡玩家作为查验目标
- 如果死亡玩家生前是恶魔，仍然会触发"是"
- 这在恶魔可能已经死亡的游戏中很有用（如小恶魔自杀传位）

#### 情况 4: 间谍 (Spy) 的影响

**间谍能力**: "可能显示为好人或任何角色"

- 说书人可以让间谍在被占卜师查验时不显示为恶魔
- 这会导致占卜师误以为间谍是好人
- 增加邪恶阵营的隐蔽性

## 3. 玩法策略

### 3.1 作为占卜师的最佳打法

#### 早期策略 (第1-2天)

**信息收集优先:**

1. **首夜选择建议**:
   - **策略A**: 选择两名发言积极的玩家（容易成为恶魔伪装目标）
   - **策略B**: 选择自己+一名可疑玩家（测试是否自己是红鲱鱼）
   - **策略C**: 随机选择两名玩家，建立基础信息网

2. **不要急于公开身份**:
   - 早期公开会成为投毒者的首要目标
   - 先私下收集2-3晚的信息
   - 等待合适时机（如有重要发现）再公开

3. **记录所有查验结果**:
   ```
   示例记录格式：
   第1夜: 玩家A + 玩家B = 否
   第2夜: 玩家C + 玩家D = 是 ← 重要！
   第3夜: 玩家C + 玩家E = 是 ← 锁定玩家C可疑
   ```

#### 中期策略 (第3-5天)

**信息三角验证:**

1. **交叉查验法**:
   - 如果"玩家X + 玩家Y = 是"
   - 下一晚查"玩家X + 玩家Z"
   - 再下一晚查"玩家Y + 玩家Z"
   - 通过三次查验锁定恶魔位置

   **示例**:
   ```
   第2夜: 座位3 + 座位7 = 是
   第3夜: 座位3 + 座位10 = 否  → 座位3大概率清白
   第4夜: 座位7 + 座位10 = 是  → 座位7高度可疑（恶魔或红鲱鱼）
   ```

2. **配合其他信息角色**:
   - 与共情者对比：如果共情者说邻居有邪恶，你查验那两个邻居
   - 与调查员对比：验证调查员给出的爪牙信息
   - 与掘墓人对比：确认被处决玩家是否为恶魔

3. **选择公开时机**:
   - **触发条件**: 当你锁定恶魔位置（连续2-3次"是"指向同一人）
   - **公开方式**: 在白天公开声明，提供完整的查验历史
   - **风险评估**: 确保至少有僧侣或其他保护角色在场

#### 后期策略 (第6天+)

**主导处决方向:**

1. **提供明确目标**:
   - 整理所有查验记录，标注高度可疑玩家
   - 在提名阶段积极推动处决可疑目标
   - 解释每次查验的逻辑链

2. **防范反击**:
   - 邪恶玩家可能会伪装占卜师混淆信息
   - 要求对方提供详细的查验历史（伪装者难以编造连贯叙述）
   - 强调自己的查验逻辑与其他信息角色的吻合度

3. **考虑红鲱鱼可能性**:
   - 如果一直查到同一人是"是"，但该玩家行为像好人
   - 考虑是否为红鲱鱼，转移查验目标
   - 与团队讨论，通过社交推理辅助判断

### 3.2 何时公开身份 vs 何时隐藏

#### 公开身份的最佳时机

**推荐公开场景:**

| 场景 | 时机 | 原因 |
|------|------|------|
| **锁定恶魔** | 第3-4天 | 连续查验指向同一人，需要推动处决 |
| **被质疑身份** | 任何时候 | 其他玩家怀疑你是邪恶，需要证明清白 |
| **配合僧侣** | 第2-3天 | 僧侣公开后，可以要求保护你 |
| **有掘墓人确认** | 第3天+ | 掘墓人可以验证被处决玩家，辅助你的信息 |

**不推荐公开场景:**

| 场景 | 原因 |
|------|------|
| **首夜/第一天** | 过早暴露，容易被投毒者针对 |
| **信息不明确** | 如果查验结果混乱，公开会降低可信度 |
| **已有其他占卜师声明** | 可能是邪恶伪装，保持隐藏收集更多信息 |
| **场上有间谍** | 间谍知道你的真实身份，公开会被针对 |

#### 隐藏身份的策略

1. **伪装成其他角色**:
   - 声称自己是士兵（无夜间能力，不会引起怀疑）
   - 或声称是外来者（降低威胁感）

2. **私密分享信息**:
   - 选择1-2名可信玩家私下透露身份
   - 让他们帮你传播查验信息，但不暴露你是占卜师

3. **延迟公开**:
   - 等到投毒者死亡或被处决后再公开
   - 或等到有明确证据时一次性爆发

### 3.3 如何有效使用能力

#### 查验优先级

**高优先级查验目标:**

1. **发言积极的玩家**: 恶魔常通过积极发言建立信任
2. **声称信息角色的玩家**: 验证他们是否撒谎
3. **投票行为异常的玩家**: 保护可疑目标、主动推动处决好人
4. **与你邻座的玩家**: 如果共情者说你邻居有邪恶

**低优先级查验目标:**

1. **沉默不发言的玩家**: 信息价值低，优先级靠后
2. **已被其他角色确认的好人**: 浪费查验机会
3. **已死亡的玩家**: 除非怀疑恶魔传位

#### 查验组合技巧

**技巧1: 概率网格法**

建立一个心理网格，将所有玩家分类：

```
清白 (No): 座位1, 座位5, 座位8
可疑 (Yes): 座位3, 座位7
未知: 座位2, 座位4, 座位6, 座位9
```

每晚选择"可疑+未知"或"未知+未知"的组合，逐步缩小范围。

**技巧2: 二分查找法**

如果有10名玩家：
- 第1夜: 查1-5号中的两人
- 第2夜: 查6-10号中的两人
- 根据结果，继续细分可疑范围

**技巧3: 死亡玩家验证法**

当有玩家被处决或夜杀后：
- 查验"死亡玩家 + 可疑活人"
- 如果得"是"，说明活人可疑
- 如果得"否"，排除活人嫌疑

## 4. 互动关系

### 4.1 与好人角色的配合

#### 与共情者 (Empath) 配合

**协同效果**: 双重验证恶魔位置

**配合策略**:
- 共情者提供"邻居中有X个邪恶"
- 占卜师查验那两个邻居
- 如果得"是"，锁定邻居中有恶魔
- 如果得"否"，说明邻居是爪牙或共情者被毒

**案例**:
```
共情者（座位5）: "我的邻居（座位4和座位6）中有1个邪恶"
占卜师第2夜: 查验座位4 + 座位6 = 是
结论: 座位4或座位6是恶魔！（概率50%）
```

#### 与掘墓人 (Undertaker) 配合

**协同效果**: 验证占卜师信息的准确性

**配合策略**:
- 如果占卜师查到某人是恶魔，推动处决
- 掘墓人当晚得知被处决者角色
- 如果掘墓人确认是恶魔，占卜师可信度提升
- 如果不是恶魔，说明占卜师可能被毒、查到红鲱鱼或隐士

#### 与僧侣 (Monk) 配合

**协同效果**: 占卜师获得保护

**配合策略**:
- 占卜师公开身份后，要求僧侣保护
- 僧侣每晚保护占卜师，防止被恶魔杀死
- 注意：僧侣无法防止投毒者下毒

#### 与杀手 (Slayer) 配合

**协同效果**: 直接击杀恶魔

**配合策略**:
- 占卜师锁定恶魔位置后，告知杀手
- 杀手白天公开使用能力击杀目标
- 如果成功，游戏立即倾向好人获胜

### 4.2 被邪恶角色克制

#### 被投毒者 (Poisoner) 克制

**威胁等级**: ⭐⭐⭐⭐⭐ (极高)

**克制原理**:
- 投毒者每晚可以给占卜师下毒
- 占卜师能力失效，接收错误信息
- 占卜师本人不知道被毒，会传播假信息误导好人

**应对策略**:
1. **不要过早公开**: 越晚公开，投毒者越难锁定你
2. **交叉验证**: 与其他信息角色对比信息，检测是否被毒
3. **观察投票行为**: 如果你的信息总是错误，考虑被毒可能性

#### 被间谍 (Spy) 迷惑

**威胁等级**: ⭐⭐⭐⭐ (高)

**克制原理**:
- 间谍看到魔典，知道谁是占卜师
- 说书人可以让间谍在被查验时不显示为邪恶
- 间谍可以伪装成占卜师，提供假信息

**应对策略**:
1. **要求详细叙述**: 如果有人声称占卜师，要求完整查验历史
2. **逻辑验证**: 检查对方的查验逻辑是否合理
3. **依赖社交推理**: 间谍难以完美伪装行为模式

#### 被猩红女巫 (Scarlet Woman) 影响

**威胁等级**: ⭐⭐ (中)

**克制原理**:
- 如果恶魔被处决，猩红女巫可能变成新恶魔
- 占卜师之前查验的结果会失效（原恶魔已死）
- 需要重新查验新恶魔

**应对策略**:
1. **识别猩红女巫**: 优先查验爪牙可疑玩家
2. **快速调整**: 恶魔死后，立即开始新的查验循环

### 4.3 对邪恶阵营的威胁

**对恶魔的威胁**: ⭐⭐⭐⭐⭐

占卜师是恶魔的头号威胁，可以直接锁定恶魔位置。恶魔通常会：
- 要求投毒者优先毒占卜师
- 或直接夜杀占卜师（如果没有僧侣保护）
- 或在白天诱导好人处决占卜师

**对爪牙的威胁**: ⭐⭐⭐

占卜师虽然不直接查验爪牙，但可以通过排除法：
- 查验大量玩家得"否"，缩小恶魔范围
- 剩余玩家中，邪恶嫌疑高的可能是爪牙

## 5. 常见误区

### 5.1 新手容易犯的错误

#### 误区1: 首夜就公开身份

**错误做法**:
```
第一天白天: "大家好，我是占卜师，昨晚查了座位3和座位5，得到'是'！"
```

**为什么错误**:
- 过早暴露身份，投毒者会立即锁定你
- 第一晚信息量不足，无法形成有效推理
- 容易被邪恶玩家针对性诱导

**正确做法**:
- 至少等到第2-3晚，收集多次查验结果
- 或等到有明确证据时再公开
- 可以私下告诉1-2名可信玩家

#### 误区2: 忽略红鲱鱼可能性

**错误做法**:
```
占卜师: "我连续三晚查到座位7都是'是'，他一定是恶魔！"
实际情况: 座位7是红鲱鱼，真恶魔是座位10
```

**为什么错误**:
- 红鲱鱼会导致持续的"是"反馈
- 单纯依赖占卜师信息会误导好人
- 忽略社交推理和其他信息角色

**正确做法**:
- 如果同一人多次出现"是"，考虑红鲱鱼
- 结合该玩家的发言、投票行为判断
- 尝试查验其他组合，排除红鲱鱼可能性

#### 误区3: 查验组合不合理

**错误做法**:
```
第1夜: 查验座位1 + 座位2 = 否
第2夜: 查验座位3 + 座位4 = 否
第3夜: 查验座位5 + 座位6 = 否
（逐对查验，效率低）
```

**为什么错误**:
- 逐对查验浪费查验机会
- 无法快速缩小恶魔范围
- 容易在查到恶魔前被毒或被杀

**正确做法**:
- 使用交叉查验法：
  ```
  第1夜: 座位1 + 座位5 = 否
  第2夜: 座位2 + 座位6 = 是 ← 锁定范围
  第3夜: 座位2 + 座位7 = 是 ← 座位2可疑！
  ```

#### 误区4: 不记录查验历史

**错误做法**:
- 仅凭记忆，无法准确回忆每晚查验结果
- 被质疑时无法提供详细证明

**为什么错误**:
- 人类记忆不可靠，容易混淆
- 邪恶玩家会要求你提供完整历史，记不清会降低可信度
- 无法进行复杂的逻辑推理

**正确做法**:
- 用纸笔记录每晚查验：玩家+玩家=结果
- 标注重要发现
- 公开时展示完整记录

#### 误区5: 过度信任自己的信息

**错误做法**:
```
占卜师: "我的信息不可能错，座位7一定是恶魔！"
（实际被投毒者连续下毒）
```

**为什么错误**:
- 占卜师可能被下毒、醉酒
- 可能查到红鲱鱼、隐士
- 盲目自信会误导整个好人团队

**正确做法**:
- 承认信息可能不准确
- 与其他角色交叉验证
- 保持开放心态，接受质疑

### 5.2 进阶陷阱

#### 陷阱1: 被邪恶玩家诱导查验方向

**场景**:
```
邪恶玩家（伪装好人）: "占卜师，你能查一下座位3和座位8吗？我觉得他们很可疑。"
实际情况: 座位3和座位8都是好人，邪恶玩家故意浪费你的查验机会
```

**应对**:
- 保持独立判断，不要完全听从他人建议
- 优先查验自己怀疑的目标
- 如果多人要求查同一人，可能是陷阱

#### 陷阱2: 与伪装占卜师混淆

**场景**:
```
第3天:
玩家A: "我是占卜师，查了座位5和座位7，得'是'"
玩家B（你）: "不，我才是占卜师，座位5和座位7是'否'！"
```

**应对**:
- 要求对方提供完整查验历史（从首夜到现在）
- 检查对方逻辑是否合理
- 让其他信息角色验证（如掘墓人、共情者）

## 6. 高级技巧

### 6.1 资深玩家的进阶玩法

#### 技巧1: 概率树推理法

**理论基础**:
每次查验可以将玩家分为三类：
- **清白**: 查验得"否"的玩家
- **可疑**: 查验得"是"的玩家
- **未知**: 未查验的玩家

**应用示例**:

```
游戏开局: 10名玩家
第1夜: 查验1+2 = 否 → 清白: {1,2}, 可疑: {}, 未知: {3,4,5,6,7,8,9,10}
第2夜: 查验3+4 = 是  → 清白: {1,2}, 可疑: {3,4}, 未知: {5,6,7,8,9,10}
第3夜: 查验3+5 = 否 → 清白: {1,2,3,5}, 可疑: {4}, 未知: {6,7,8,9,10}
结论: 座位4极度可疑（恶魔或红鲱鱼）
```

**进阶策略**:
- 使用概率计算：如果可疑池有3人，每人是恶魔的概率约33%（考虑红鲱鱼）
- 优先处决概率最高的目标

#### 技巧2: 反向心理战

**策略**: 故意公开错误信息，诱导邪恶暴露

**应用示例**:
```
第4天白天（占卜师公开）:
占卜师: "我查了座位7和座位9，得'否'，他们是清白的。"
（实际查验结果: 座位7+座位9 = 是）

邪恶玩家可能放松警惕，以为占卜师被毒或信息错误。
第5天，占卜师突然公开真实信息，推动处决座位7。
```

**风险**:
- 可能误导好人队友
- 需要高超的社交技巧和时机把握
- 仅适合资深玩家

#### 技巧3: 红鲱鱼自我识别法

**策略**: 通过查验自己判断是否为红鲱鱼

**应用示例**:
```
第1夜: 查验 自己 + 玩家A = 是
第2夜: 查验 自己 + 玩家B = 是
第3夜: 查验 自己 + 玩家C = 是
结论: 自己大概率是红鲱鱼！（除非恶魔恰好是ABC之一）
```

**优势**:
- 确认自己是红鲱鱼后，可以排除所有查验结果
- 告知好人队友"忽略我的所有'是'查验"
- 转而通过"否"查验排除好人

#### 技巧4: 延迟公开爆发流

**策略**: 隐藏到后期，一次性公开大量信息

**应用示例**:
```
第1-5天: 完全不公开，伪装成普通玩家
第6天: 突然公开，提供5晚的详细查验记录
效果:
- 投毒者无法提前针对
- 信息量巨大，可以直接锁定恶魔
- 好人团队可以立即处决
```

**适用场景**:
- 场上有僧侣或其他保护角色
- 游戏进入中后期，急需明确目标
- 有其他占卜师声明（可能是邪恶伪装），需要澄清

### 6.2 与说书人的博弈

#### 理解说书人的平衡考量

说书人在处理占卜师时需要平衡游戏：

**说书人可能的操作**:
1. **选择强势红鲱鱼**: 如果占卜师很强，可能选活跃玩家为红鲱鱼
2. **适度给予真实信息**: 如果好人劣势，可能减少误导
3. **利用隐士**: 让隐士在关键查验中显示为恶魔

**玩家应对**:
- 理解说书人不是敌人，而是游戏平衡者
- 不要抱怨"说书人针对我"，这是游戏机制
- 通过社交推理弥补信息不准确的部分

#### 识别说书人的暗示

**场景1**: 说书人给信息时的微表情

某些说书人可能在给关键信息时有微妙表情变化：
- 点头特别坚定 → 可能是真恶魔
- 点头犹豫 → 可能是红鲱鱼或隐士

**注意**: 这取决于说书人的个人习惯，不可全信。

**场景2**: 说书人的查验速度

- 立即点头/摇头 → 结果明确
- 停顿思考后点头 → 可能在权衡是否让隐士显示为恶魔

## 7. 说书人指引

### 7.1 如何公正处理占卜师

#### 红鲱鱼选择原则

**推荐选择标准**:

| 玩家人数 | 红鲱鱼选择建议 | 理由 |
|---------|--------------|------|
| 5-7人 | 占卜师自己 或 活跃镇民 | 小局游戏信息密集，需要增加混淆 |
| 8-10人 | 中等活跃玩家 | 平衡游戏，不过度削弱占卜师 |
| 11-15人 | 镇民角色 | 大局游戏信息分散，红鲱鱼影响相对较小 |

**不推荐选择**:
- ❌ 外来者（圣徒、酒鬼等）：他们本身已是好人劣势
- ❌ 完全沉默的玩家：占卜师不太会查验他们，浪费红鲱鱼机制
- ❌ 恶魔或爪牙：违反规则，红鲱鱼必须是好人

#### 处理隐士的查验

**场景**: 占卜师查验包含隐士的组合

**说书人选择**:

1. **让隐士显示为恶魔** (点头"是")
   - 适用场景: 好人信息过于准确，需要增加混淆
   - 效果: 占卜师误以为找到恶魔

2. **让隐士显示为外来者** (摇头"否")
   - 适用场景: 好人已经很困惑，给予准确信息
   - 效果: 占卜师排除隐士

**平衡建议**:
- 隐士不应每次都显示为恶魔（过于明显）
- 建议50-70%的概率显示为恶魔
- 根据游戏局势灵活调整

#### 处理被毒/醉的占卜师

**场景**: 占卜师被投毒者下毒

**说书人选择**:

1. **给完全错误的信息**:
   - 真实: 座位3 + 座位7 = 是（座位3是恶魔）
   - 告知: 摇头"否"
   - 效果: 占卜师误以为两人清白

2. **给部分正确的信息**:
   - 真实: 座位5 + 座位8 = 否
   - 告知: 点头"是"
   - 效果: 占卜师误以为有恶魔，浪费时间调查

3. **给随机信息**:
   - 随机点头或摇头
   - 效果: 增加不可预测性

**推荐策略**:
- 前期被毒：给错误信息，误导好人
- 后期被毒：适度给真实信息，避免游戏过于倾向邪恶

### 7.2 如何选择红鲱鱼

#### 情况1: 占卜师新手玩家

**策略**: 选择温和的红鲱鱼

- 选择非关键位置的镇民
- 或选择占卜师自己（帮助他们学习机制）
- 避免选择核心信息角色

#### 情况2: 占卜师资深玩家

**策略**: 选择挑战性高的红鲱鱼

- 选择活跃、发言逻辑强的镇民
- 或选择与恶魔行为相似的玩家
- 增加游戏难度和趣味性

#### 情况3: 游戏人数少（5-7人）

**策略**: 红鲱鱼影响更大

- 建议选择占卜师本人
- 或选择外来者以外的镇民
- 小心不要过度削弱好人阵营

#### 情况4: 游戏人数多（12-15人）

**策略**: 红鲱鱼影响相对较小

- 可以选择任何好人玩家
- 甚至可以选择"无红鲱鱼"（罕见，但可行）
- 大局游戏信息已经足够复杂

### 7.3 特殊场景裁决

#### 场景1: 占卜师查验两个邪恶玩家

**示例**: 查验小恶魔 + 投毒者

**说书人裁决**: 点头"是"（因为有恶魔）

**注意**: 占卜师只会知道"有恶魔"，不会知道两个都是邪恶

#### 场景2: 占卜师查验红鲱鱼 + 隐士

**示例**: 查验红鲱鱼 + 隐士（隐士可能显示为恶魔）

**说书人选择**:

1. **选项A**: 点头"是"（红鲱鱼触发）
2. **选项B**: 点头"是"（隐士显示为恶魔）
3. **选项C**: 摇头"否"（隐士显示为外来者，但红鲱鱼被忽略）

**推荐**: 选项A或B，因为至少有一个"干扰因素"

#### 场景3: 占卜师死后复活

**示例**: 占卜师被杀，教授复活他

**说书人裁决**:
- 复活后，占卜师能力恢复
- 继续每晚醒来查验
- 红鲱鱼保持不变（除非红鲱鱼已死或变成邪恶）

#### 场景4: 占卜师查验旅行者

**说书人裁决**:
- 旅行者不是恶魔（除非有特殊邪恶旅行者）
- 通常会得"否"
- 旅行者不能成为红鲱鱼（因为他们可能随时离开游戏）

## 8. 经典案例

### 案例1: 红鲱鱼陷阱突破

**背景**: 9人局，Trouble Brewing剧本

**玩家配置**:
- 座位1: 洗衣妇
- 座位2: **占卜师**（你）
- 座位3: 士兵
- 座位4: 管家（外来者）
- 座位5: 僧侣
- 座位6: **投毒者**（爪牙）
- 座位7: 共情者（**红鲱鱼**）
- 座位8: 厨师
- 座位9: **小恶魔**（恶魔）

**查验记录**:

```
第1夜: 座位7 + 座位9 = 是 ← （座位7是红鲱鱼，座位9是恶魔）
第2夜: 座位7 + 座位8 = 是 ← （座位7持续触发）
第3夜: 座位7 + 座位1 = 是 ← （座位7持续触发）
```

**占卜师思考**:
> "座位7连续三次都是'是'，他大概率是红鲱鱼或恶魔。但座位7公开声称共情者，且信息与洗衣妇吻合，行为像好人。我应该考虑红鲱鱼可能性。"

**突破策略**:

```
第4夜: 座位9 + 座位6 = 是 ← （座位9恶魔 + 座位6投毒者）
第5夜: 座位9 + 座位5 = 是 ← （座位9恶魔，座位5僧侣是"否"，但被投毒）
```

**结论**:
- 座位9在多次查验中都出现"是"
- 结合投票行为（座位9保护可疑玩家），锁定座位9是恶魔
- 第6天推动处决座位9，好人获胜！

**关键教训**:
- 不要盲目相信第一次"是"的结果
- 通过交叉查验排除红鲱鱼
- 结合社交推理和其他信息角色


**占卜师察觉异常**:
> "我查了座位10两次都是'否'，但共情者说座位9和座位11（座位10的邻居）有邪恶。这不对劲！我可能被毒了。"

**第4天白天**:
```
占卜师（私下对僧侣）: "我怀疑自己被投毒者下毒了，我的信息可能不准。我们需要先处决投毒者。"
```

**第5天白天** (投毒者被处决):
```
第5夜: 座位10 + 座位6 = 是 ← （投毒者已死，恢复正常）
结论: 座位10是恶魔！
```

**第6天**: 推动处决座位10，好人获胜。

**关键教训**:
- 与其他信息角色交叉验证，检测是否被毒
- 优先处决投毒者，恢复信息准确性
- 私下沟通，避免公开暴露被毒

### 案例3: 双占卜师对决

**背景**: 12人局，邪恶玩家伪装占卜师混淆视听

**玩家配置**:
- 座位4: **真占卜师**（你）
- 座位8: **间谍**（伪装占卜师）
- 座位12: 小恶魔

**第3天白天发生**:

**间谍（座位8）公开声明**:
> "我是占卜师！我查了座位4和座位5，都是好人（'否'）。昨晚查了座位9和座位10，得到'是'，其中一个是恶魔！"

**真占卜师（座位4）立即反击**:
> "不对！我才是真占卜师。我的查验记录是：
> - 第1夜: 座位8 + 座位12 = 是
> - 第2夜: 座位8 + 座位3 = 否
> - 第3夜: 座位12 + 座位7 = 是
> 根据我的查验，座位12极度可疑（恶魔或红鲱鱼）！"

**好人团队的选择**:
- 需要判断谁是真占卜师

**验证方法**:

1. **掘墓人验证**:
   - 如果处决其中一人，掘墓人会得知真实角色
   - 风险：如果处决真占卜师，好人损失惨重

2. **逻辑验证**:
   - 要求两人详细解释查验逻辑
   - 间谍难以编造完全合理的查验历史

3. **共情者验证**:
   - 共情者检查座位8的邻居是否有邪恶
   - 如果座位8邻居有邪恶，座位8可能是间谍

**结果**:
- 共情者确认座位8邻居有邪恶
- 好人团队相信座位4（真占卜师）
- 第4天处决座位12（小恶魔），好人获胜

**关键教训**:
- 伪装占卜师需要提供详细、合理的查验历史
- 真占卜师应该记录完整信息，便于证明
- 依赖其他信息角色交叉验证

## 9. 总结与建议

### 占卜师的核心价值

**优势**:
- ⭐ 每晚获取信息，持续性强
- ⭐ 可以直接查验恶魔，威胁巨大
- ⭐ 信息相对明确（是/否），易于理解
- ⭐ 在大局游戏中尤其强力

**劣势**:
- ❌ 容易被投毒者针对
- ❌ 红鲱鱼机制增加误导可能性
- ❌ 需要多晚查验才能锁定目标
- ❌ 过早公开会成为恶魔击杀目标

### 给新手占卜师的建议

1. **不要急于公开**: 至少等到第2-3晚
2. **记录所有查验**: 用纸笔记录，便于回忆
3. **考虑红鲱鱼**: 如果同一人多次"是"，可能是红鲱鱼
4. **与队友配合**: 结合共情者、掘墓人等信息
5. **保持谦逊**: 承认信息可能不准确，接受质疑

### 给资深占卜师的建议

1. **使用概率推理**: 建立概率网格，量化嫌疑
2. **掌握心理战**: 适时隐瞒或公开信息
3. **识别被毒**: 交叉验证信息，检测异常
4. **主导节奏**: 在关键时刻引导好人处决方向
5. **与说书人博弈**: 理解说书人的平衡考量，适应游戏

### 给说书人的建议

1. **平衡游戏**: 红鲱鱼选择应根据玩家水平调整
2. **适度误导**: 不要过度削弱占卜师，保持游戏趣味
3. **公正裁决**: 处理隐士、被毒等情况时保持一致性
4. **观察局势**: 根据好人/邪恶优势动态调整信息准确度
