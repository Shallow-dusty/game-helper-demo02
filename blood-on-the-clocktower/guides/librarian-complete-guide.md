# 图书管理员（Librarian）深度完整手册

## 📚 角色基础信息

### Role Identity / 角色身份
- **英文名称**: Librarian
- **中文名称**: 图书管理员
- **阵营 (Team)**: 善良 (Good)
- **类型 (Type)**: 镇民 (Townsfolk)
- **版本 (Edition)**: Trouble Brewing (初酿风波)

### Ability Description / 能力描述

**英文原文**:
> "You start knowing that 1 of 2 players is a particular Outsider. (Or that zero are in play.)"

**中文翻译**:
> "游戏开始时，你会得知2名玩家中有1人是特定的外来者。（或者得知场上没有外来者。）"

---

## ⚙️ 能力机制详解

### Ability Mechanics / 运作机制

#### First Night Only / 仅限第一夜
图书管理员的能力**仅在第一夜触发一次**，之后不再获得新信息。

#### Information Received / 获得的信息

说书人会在第一夜唤醒图书管理员，并提供以下信息之一：

1. **标准信息**: 指向两名玩家，并展示一个外来者角色标记
   - 含义：这两人中**有且仅有一人**是该外来者
   - 你**不知道具体是哪一个**

2. **零外来者信息**: 展示数字"0"
   - 含义：本局游戏**没有任何外来者**

#### Special Rules / 特殊规则

##### 关于醉酒者 (The Drunk)
这是最重要的机制细节：

- 如果被指示的外来者是**醉酒者 (Drunk)**
- 图书管理员看到的是 **"Drunk"** 角色标记
- **而不是**醉酒者自己以为的角色

**示例**:
```
场景：Alice 是醉酒者，她以为自己是占卜师
说书人告诉图书管理员：
- 指向 Alice 和 Bob
- 展示 "Drunk" 标记
✓ 正确：图书管理员得知其中一人是醉酒者
✗ 错误：图书管理员不会看到"占卜师"
```

##### 关于中毒/醉酒状态
如果图书管理员自己是**醉酒者或被下毒**：
- 可能收到**完全错误的信息**
- 可能被告知不存在的外来者
- 可能被告知错误的玩家组合

## 🎯 玩法策略

### As the Librarian / 作为图书管理员

#### 核心目标
1. **识别真实的外来者** - 从两名候选人中找出真正的外来者
2. **保护有价值的信息** - 决定何时公开、向谁公开
3. **利用确认的善良票** - 即使外来者能力负面，也是确认的善良玩家

### Opening Strategy / 开局策略

#### Scenario 1: 收到外来者信息

**Day 1 Actions / 第一天行动**:

1. **观察两名候选人的行为**
   - 谁主动认领外来者身份？
   - 谁的发言更符合该外来者的视角？
   - 谁可能在隐藏什么？

2. **不要立即公开所有信息**
   ```
   ❌ 错误做法：
   "我是图书管理员，Alice和Bob中有一个是圣徒！"

   ✓ 更好做法：
   私下找可信玩家交流，观察候选人反应后再决定
   ```

3. **建立私密联盟**
   - 找1-2个看起来可信的玩家
   - 私下分享信息
   - 交叉验证彼此的信息

#### Scenario 2: 收到"0"信息

这是**极其强大**的信息！

**含义分析**:
- ✓ 场上没有男爵 (Baron)
- ✓ 任何宣称是外来者的玩家**极有可能是邪恶阵营**
- ✓ 镇民/仆从数量标准（没有额外外来者）

**行动建议**:
```
Day 1-2: 保持沉默，观察谁会宣称外来者
Day 2-3: 当有人宣称外来者时，准备揭穿
Day 3+: 公开"0"信息，质疑所有外来者宣称
```

**警告**: 确保自己不是醉酒者或中毒！

### When to Reveal / 何时公开身份

#### 公开时机选择

| 时机 | 优势 | 劣势 | 推荐场景 |
|------|------|------|----------|
| **第一天立即公开** | 快速建立信任 | 失去信息优势，成为邪恶目标 | 收到"0"且急需建立领导地位 |
| **第二天有选择公开** | 观察后做出明智决策 | 可能显得可疑 | 已识别出外来者身份 |
| **保持隐藏** | 保持灵活性，避免成为目标 | 无法充分利用信息 | 收到模糊信息或怀疑自己中毒 |
| **临终公开** | 死前传递关键信息 | 浪费早期价值 | 邪恶势力强大，需保命 |

#### 最佳实践
```
推荐流程：
1. Day 1: 私下接触2-3个玩家，建立小圈子
2. Day 2: 根据外来者候选人表现，缩小范围
3. Day 3: 有选择地公开部分信息
4. Day 4+: 根据局势决定是否完全公开
```

### Advanced Techniques / 高级技巧

#### 技巧1: 钓鱼执法
```
情景：你收到"0"外来者信息

步骤1: 不公开你的角色
步骤2: 在公开讨论中说："我怀疑场上有男爵"
步骤3: 观察谁会跳出来宣称外来者
步骤4: 等多人宣称后，揭露"0"信息，一网打尽
```

#### 技巧2: 双重验证
```
你得知: Alice/Bob 中一人是圣徒

行动:
1. 找到占卜师，询问他们是否查验过这两人
2. 找到洗衣妇，看是否有相关信息
3. 找到调查员，交叉验证
4. 综合多方信息，提高准确度
```

#### 技巧3: 假信息诱饵
```
场景: 你怀疑Carol是恶魔，但缺乏证据

策略:
1. 公开虚假信息："Carol/Dave 中一人是酒鬼"
2. 观察Carol的反应：
   - 如果她疯狂否认 → 可能确实是恶魔
   - 如果她认领酒鬼 → 可能是善良被误导
3. 后期澄清真实信息，但已获得Carol的行为数据
```

**警告**: 这是高风险策略，可能破坏团队信任！

## 🤝 互动关系

### Synergies / 配合角色

#### 强力组合

**图书管理员 + 占卜师 (Fortune Teller)**
- 占卜师验证你指出的两名候选人
- 快速确认哪一个是外来者
- 信息价值倍增

**图书管理员 + 洗衣妇 (Washerwoman)**
- 洗衣妇可能看到相同的外来者
- 交叉验证，确认彼此真实性
- 建立强大的信息网络

**图书管理员 + 共情者 (Empath)**
- 共情者坐在候选人旁边
- 通过邻居邪恶数量推断外来者身份
- 间接确认信息

**图书管理员 + 送葬者 (Undertaker)**
- 外来者死亡后，送葬者确认身份
- 验证你的信息准确性
- 揭露可能的中毒/醉酒

#### 需谨慎的组合

**图书管理员 + 间谍 (Spy)**
- 间谍可能假装是你指出的外来者
- 间谍能看到说书人给你的信息
- 高度警惕虚假认领

### Counter Relationships / 克制关系

#### 被以下角色克制

**中毒者 (Poisoner)**
- 第一夜被下毒 → 收到错误信息
- 你的所有推理都基于错误前提
- 难以自我察觉

**间谍 (Spy)**
- 知道你的信息，可以完美伪装
- 可能假扮成你指出的外来者
- 利用你的信任获取情报

**男爵 (Baron)**
- 增加外来者数量
- 让你的信息更难验证
- 多个外来者宣称造成混乱

#### 克制以下角色

**恶魔伪装外来者**
- "0"信息直接揭露
- 快速识别伪装者
- 缩小恶魔范围

**爪牙伪装外来者**
- 同上，快速识别
- 减少邪恶阵营隐藏空间

## ⚠️ 常见误区

### Mistakes to Avoid / 新手易犯错误

#### 误区1: 第一天立即全盘公开
```
错误示范:
"大家好！我是图书管理员！Alice和Bob中有一个是圣徒！"

为什么错误:
❌ 立即成为恶魔/中毒者的目标
❌ 失去观察候选人反应的机会
❌ 邪恶阵营可以提前布局应对
❌ 无法利用信息优势建立私密联盟
```

#### 误区2: 过度信任自己的信息
```
错误思维:
"我看到的肯定是真的，我不可能被下毒！"

现实:
✓ 你可能是醉酒者
✓ 你可能第一夜被下毒
✓ 说书人可能给了"红色警报"信息（平衡机制）

正确做法:
持续寻找验证信息的方式，保持质疑
```

#### 误区3: 忽视"0"信息的威力
```
错误做法:
收到"0" → "哦，没啥信息" → 不作为

正确认识:
"0" = 极强信息！
→ 可以揭穿所有外来者伪装
→ 确认没有男爵
→ 掌握局势主动权
```

#### 误区4: 对外来者候选人过度敌对
```
错误行为:
"Bob你一定是酒鬼！你表现得太蠢了！"

问题:
❌ 即使Bob是外来者，他也是善良阵营
❌ 过度敌对让真正的善良玩家反感
❌ 让邪恶阵营渔翁得利

正确做法:
友善地引导，私下交流，建立信任
```

#### 误区5: 死守错误推断
```
场景:
你认定Alice是外来者，但后续信息显示可能是Bob

错误:
"不可能！肯定是Alice！"（认知固化）

正确:
保持开放思维，根据新信息调整判断
```

### How to Avoid / 避免方法

1. **建立验证机制**
   - 寻找其他信息角色交叉验证
   - 观察候选人长期行为
   - 不要只依赖第一印象

2. **考虑被干扰可能性**
   - 始终假设可能中毒/醉酒
   - 寻找证据证明信息真实性
   - 如果信息与局势严重不符，考虑自己状态异常

3. **灵活的公开策略**
   - 不要预设"必须第一天公开"
   - 根据局势动态调整
   - 有时隐藏更有价值

## 🎓 高级玩法

### Expert Strategies / 资深玩家技巧

#### 策略1: 信息分层披露
```
原理: 不要一次性公开所有信息，而是分阶段释放

Day 1: "我有一些关于外来者的信息"
       → 观察谁紧张，谁好奇

Day 2: "我知道某两人中有一个外来者"
       → 看谁开始布局应对

Day 3: "是Alice和Bob"
       → 观察两人反应差异

Day 4: "是圣徒"
       → 最终信息，此时已收集大量行为数据
```

#### 策略2: 利用负面信息
```
场景: 你得知有"酒鬼"或"隐士"等负面外来者

高级玩法:
1. 不立即揭露具体角色
2. 让候选人自由发挥
3. 观察谁的行为异常（可能是真酒鬼不自知）
4. 在关键投票前才揭露，避免分裂
```

#### 策略3: 制造信息迷雾
```
适用场景: 你怀疑自己被下毒

操作:
1. 公开一半真信息，一半假信息
2. 观察邪恶阵营的应对：
   - 如果他们按假信息布局 → 证明你确实被下毒
   - 如果他们按真信息应对 → 你的信息可能准确
3. 后期根据观察调整策略
```

#### 策略4: "双重身份"博弈
```
高级技巧: 先伪装成其他角色

Day 1-2: 暗示自己是"共情者"
       → 邪恶阵营放松警惕

Day 3: 突然公开图书管理员身份和信息
       → 打乱邪恶阵营部署
       → 此时他们已浪费时间针对"共情者"
```

### Mind Games / 心理战术

#### 读懂微表情
```
当你暗示有外来者信息时，观察:
- 谁的眼神飘忽？（可能心虚）
- 谁过度镇定？（可能是间谍，已知信息）
- 谁立即主动接话？（可能想控制话题）
```

#### 压力测试
```
技巧: 故意给错误压力

对Alice说: "你表现得像外来者"
        （即使你知道是Bob）

目的:
- 测试Alice的反应（真善良会如何辩护？）
- 看Bob是否会"帮忙"（暴露知情）
- 评估整体局势
```

## 🎭 说书人指引

### For Storytellers / 给说书人的建议

#### 选择哪两名玩家？

**原则1: 平衡性**
```
✓ 好的选择:
- 两名都是活跃玩家（增加互动）
- 一名新手+一名老手（教学机会）
- 两名性格反差大的玩家（增加戏剧性）

✗ 避免:
- 两名完全沉默的玩家（信息浪费）
- 一名玩家会立即暴露身份（信息贬值）
```

**原则2: 戏剧性**
```
优秀选择示例:
- 圣徒 + 一名邪恶爪牙伪装
  → 真假圣徒对决

- 酒鬼 + 实际镇民
  → 酒鬼可能疯狂坚持错误信息

- 隐士 + 社交核心玩家
  → 隐士机制创造有趣互动
```

#### 关于"醉酒者"的处理

**关键规则**: 图书管理员看到的是 **"Drunk"** 标记

```
场景设置:
- Alice 是醉酒者，以为自己是调查员
- Bob 是普通镇民

说书人操作:
第一夜:
1. 唤醒图书管理员
2. 指向 Alice 和 Bob
3. 展示 "Drunk" 角色标记
4. 图书管理员睡下

✓ 正确
✗ 不要展示"调查员"
✗ 不要给出模糊信息
```

#### 给"0"信息的时机

**何时展示"0"？**
```
标准情况:
- 场上确实没有外来者
- 没有男爵
- 标准镇民/仆从配置

特殊情况:
- 如果图书管理员是醉酒者
  → 可以给"0"（即使有外来者）
  → 制造混乱

- 如果图书管理员被第一夜下毒
  → 同上
```

#### 平衡游戏体验

**如果图书管理员信息过强**:
```
合法调整手段:
1. 选择不太明显的外来者候选人
2. 确保邪恶阵营有间谍（可应对）
3. 考虑第一夜下毒（如有中毒者）

禁止手段:
✗ 故意给错误信息（除非醉酒/中毒）
✗ 改变已展示的信息
✗ 事后否认玩家看到的内容
```

**如果图书管理员信息过弱**:
```
帮助手段:
1. 选择容易识别的外来者（如管家）
2. 选择表演型玩家作为候选人
3. 确保外来者会主动使用能力

避免:
✗ 直接暗示答案
✗ 给予额外信息
✗ 偏袒任何阵营
```

## 📖 经典案例分析

### Case Study 1: "零外来者"的完美利用

**背景设定**:
- 10人局
- 图书管理员收到"0"
- Day 2，三名玩家宣称外来者

**玩家操作**:
```
Day 1:
图书管理员: 保持沉默，观察

Day 2:
- Alice宣称: "我是圣徒"
- Bob宣称: "我是酒鬼"
- Carol宣称: "我是管家，服侍Dave"

图书管理员: 继续观察，不表态

Day 3:
图书管理员: 公开身份
"我是图书管理员，第一夜收到'0'，
 场上没有外来者！
 Alice、Bob、Carol都在撒谎！"

结果:
- 全镇震惊
- 经过质询，Bob承认是爪牙
- Alice实际是恶魔，伪装失败
- Carol慌乱中暴露破绽
- 善良阵营大胜
```

**关键要点**:
✓ 耐心等待邪恶阵营暴露
✓ 在关键时刻一击必杀
✓ 用确凿信息建立领导力

### Case Study 2: 醉酒者陷阱

**背景设定**:
- 图书管理员得知: Alice/Bob 其一是"酒鬼"
- Alice 实际是酒鬼，以为自己是占卜师
- Bob 是真占卜师

**复杂之处**:
```
Day 2:
Alice公开: "我是占卜师，验到Carol和Dave，
          其中Carol是恶魔！"

Bob也公开: "我才是占卜师！Alice在撒谎！"

全镇混乱: 两个占卜师！

图书管理员困境:
- 我知道其一是酒鬼
- 但哪个是真占卜师？
- 酒鬼会坚信自己的信息
```

**高手操作**:
```
图书管理员策略:

1. 不立即公开身份
2. 私下问Alice: "你第一夜验谁了？"
3. 私下问Bob: "你第一夜验谁了？"
4. 对比回答细节：
   - 谁的回答更像真信息？
   - 谁的行为符合角色逻辑？

Day 3:
图书管理员公开:
"我是图书管理员，Alice和Bob中一人是酒鬼。
 根据行为分析，我认为Alice是酒鬼。
 Bob，请继续发挥占卜师作用。"

结果:
- 正确识别酒鬼
- 保护真占卜师
- Alice（酒鬼）虽困惑但理解
- 团队信息清晰化
```

**学习要点**:
✓ 酒鬼信息需要细致甄别
✓ 不要急于下结论
✓ 通过行为模式判断
✓ 温和处理（酒鬼也是善良）

### Case Study 3: 被下毒的觉醒

**背景设定**:
- 图书管理员第一夜被中毒者下毒
- 收到错误信息: Eve/Frank 其一是隐士
- 实际: Eve和Frank都是普通镇民，真隐士是George

**错误信息的连锁反应**:
```
Day 1-2:
图书管理员基于错误信息行动
- 重点关注Eve/Frank
- 忽视真正的隐士George

Day 3:
George（真隐士）死亡
送葬者公开: "George是隐士"

图书管理员震惊:
"等等... 我的信息说是Eve/Frank！"

觉醒时刻:
意识到自己被下毒了！
```

**高手应对**:
```
立即调整策略:

1. 公开承认:
   "我是图书管理员，但我的信息是错的，
    我第一夜被下毒了！"

2. 关键推理:
   "谁是中毒者？谁第一夜下毒了我？
    这个人知道我是图书管理员！"

3. 反向利用:
   "中毒者知道我的身份，
    他们可能坐在我附近，
    或者通过恶魔信息得知。"

4. 缩小范围:
   结合其他信息，锁定可疑中毒者

结果:
- 虽然初期信息错误
- 但通过"被下毒"这一事实
- 反向推理出中毒者身份
- 将劣势转化为优势
```

**深刻教训**:
✓ 错误信息也是信息
✓ 被干扰本身就是线索
✓ 保持灵活，及时调整
✓ 承认错误，重建信任

### Case Study 4: 间谍的完美伪装

**背景设定**:
- 邪恶阵营有间谍
- 图书管理员得知: Harry/Iris 其一是圣徒
- 间谍看到说书人给图书管理员的信息
- 间谍坐在Harry的位置（他是间谍，不是圣徒）

**间谍的操作**:
```
Day 1:
间谍Harry第一时间公开:
"我是圣徒，如果被处决，善良输！"
（完美利用图书管理员的信息）

图书管理员想法:
"太好了！Harry主动认领了，肯定是他！"
✓ 确认Harry是圣徒
✓ 那么Iris就是普通镇民

Day 2-4:
- 全镇保护Harry（实际是间谍）
- Iris被忽视
- 间谍Harry疯狂收集情报
- 邪恶阵营掌握主动权
```

**如何识破？**
```
高手图书管理员的警觉:

Day 2:
观察到异常:
"Harry为什么这么急于认领圣徒？
 真圣徒通常会更谨慎..."

验证测试:
1. 找共情者: "Harry邻居有邪恶吗？"
2. 找占卜师: "验证一下Harry"
3. 观察Harry的投票: "是否总是善良阵营投票？"

Day 3:
发现疑点:
- Harry的投票偶尔偏向邪恶
- 共情者暗示Harry邻居有问题
- Harry对某些信息反应过度

Day 4:
勇敢质疑:
"我重新审视了信息，
 我现在认为Iris才是真圣徒，
 Harry可能是间谍！"

投票处决Harry → 证实是间谍 → 挽救局势
```

**高级经验**:
✓ 永远保持质疑
✓ 即使信息匹配，也观察行为
✓ 间谍会利用你的信息
✓ 交叉验证是关键

## 🔧 实战技巧总结

### Quick Reference / 速查表

#### 开局检查清单
- [ ] 确认收到的信息（外来者类型 or "0"）
- [ ] 记住两名候选人（如适用）
- [ ] 评估自己可能被下毒/醉酒的概率
- [ ] 制定第一天的发言策略
- [ ] 识别潜在的盟友

#### Day 1 行动指南
- [ ] 观察候选人的发言和行为
- [ ] 识别其他信息角色
- [ ] 建立1-2个私密联系
- [ ] 决定是否当天公开（通常不建议）
- [ ] 收集其他玩家的信息

#### Day 2+ 持续行动
- [ ] 交叉验证你的信息
- [ ] 根据新信息调整判断
- [ ] 选择合适时机公开身份
- [ ] 领导讨论（如果公开）
- [ ] 保护善良阵营

#### 投票前检查
- [ ] 你的信息是否被验证？
- [ ] 是否有矛盾需要解释？
- [ ] 外来者候选人的表现如何？
- [ ] 其他信息是否支持你的判断？
- [ ] 是否有被干扰的迹象？

### 关键原则

1. **信息是武器，也是负担**
   - 妥善保管
   - 择时释放
   - 交叉验证

2. **外来者是盟友**
   - 即使能力负面
   - 也是确认的善良票
   - 友善对待

3. **保持怀疑**
   - 你可能被下毒
   - 间谍可能伪装
   - 信息可能过时

4. **灵活应变**
   - 没有固定公式
   - 根据局势调整
   - 承认错误，快速修正

### 推荐学习路径

**进阶阶段** (6-15局):
- 实验不同公开策略
- 学习交叉验证技巧
- 理解被干扰的识别

**高手阶段** (16+局):
- 心理战术运用
- 复杂局势分析
- 说书人视角思考

### 常见问题 FAQ

**Q1: 如果我收到的两个候选人都宣称是外来者怎么办？**
```
A: 这是好事！说明:
   1. 其中一个说真话（你的信息准确）
   2. 另一个在撒谎（找到邪恶线索）
   3. 通过行为细节判断谁是真的
```

**Q2: 我怎么知道自己是不是被下毒了？**
```
A: 寻找矛盾:
   - 如果你的信息与多个可信来源冲突
   - 如果外来者候选人的表现都不像外来者
   - 如果收到"0"但明显有外来者存在
   → 考虑被干扰可能性
```

**Q3: 收到"酒鬼"信息后，如何判断谁是酒鬼？**
```
A: 观察关键点:
   - 谁的信息过于荒谬？
   - 谁坚持明显错误的判断？
   - 谁的角色宣称与其他人冲突？
   - 真酒鬼通常不自知，表现"真诚的错误"
```

**Q4: 图书管理员是否应该第一天就公开？**
```
A: 取决于信息:

   收到"0" → 可考虑早公开（强力信息）
   收到外来者信息 → 建议晚公开（需观察）
   怀疑被下毒 → 谨慎公开（先验证）

   没有绝对规则，根据局势判断
```

**Q5: 如果外来者是圣徒，我该怎么办？**
```
A: 特别小心:
   1. 不要公开指出谁是圣徒（保护他们）
   2. 私下联系圣徒，建立信任
   3. 确保他们理解不要被提名
   4. 在投票时保护圣徒
```

## 🎯 总结

### 核心要点回顾

**图书管理员的本质**:
- 第一夜的信息角色
- 专注于外来者识别
- 信息型领导者

**成功的关键**:
1. 耐心观察
2. 交叉验证
3. 灵活公开
4. 保持质疑
5. 团队协作

**避免的陷阱**:
1. 过早公开
2. 盲目信任
3. 忽视"0"
4. 敌对外来者
5. 认知固化

### 最后的建议

图书管理员是一个**看似简单，实则深奥**的角色。

- 新手可以快速上手
- 高手可以玩出花样
- 每局都有新的挑战

**最重要的一点**:
> 记住你是在帮助善良阵营，而不是在证明自己正确。
> 有时承认错误，比坚持错误更有价值。

祝你在托儿河镇的冒险中，找到真相，守护善良！
