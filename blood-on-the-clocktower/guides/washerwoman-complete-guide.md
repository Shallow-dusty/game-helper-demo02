## 1. 基础信息

### 1.1 角色定位

**Washerwoman（洗衣妇）** 是血染钟楼基础剧本 **Trouble Brewing（暗流涌动）** 中的一个镇民（Townsfolk）角色。作为首夜信息角色，洗衣妇在游戏开始时就能获得关键情报，是善良阵营建立信息网络的重要基石。

**角色类型**：镇民 Townsfolk
**剧本归属**：Trouble Brewing（暗流涌动）
**复杂度**：★☆☆☆☆（新手友好）
**影响力**：★★★☆☆（中等，取决于使用方式）

### 1.2 能力描述

> **"你在首个夜晚会得知两名玩家，其中一位是特定的镇民角色。"**

用更通俗的话来说：游戏第一个夜晚，说书人（Storyteller，也就是主持游戏的人）会告诉你两个玩家的名字，并告诉你一个镇民角色的名称。这两个玩家中，有一个人就是那个镇民角色。

**举例**：
- 说书人对你说："3号玩家和7号玩家，其中一位是僧侣（Monk）。"
- 这意味着：3号和7号中，必有一人是僧侣，另一人不是僧侣（可能是其他镇民、外来者、爪牙或恶魔）

### 1.3 剧本生态位置

在 Trouble Brewing 剧本中，洗衣妇与其他首夜信息角色形成了善良阵营的信息骨架：

**首夜信息角色三人组**：
- **Washerwoman（洗衣妇）**：确认镇民身份
- **Librarian（图书管理员）**：确认外来者身份
- **Investigator（调查员）**：确认爪牙身份

这三个角色互相补充，共同构建了善良阵营的初始信息网。洗衣妇专注于识别镇民，这在游戏中尤为重要，因为：

1. **镇民占多数**：标准局中镇民数量最多（13人局有7个镇民）
2. **强力角色确认**：可以直接确认僧侣、占卜师、送葬者等关键角色
3. **建立信任基础**：确认的镇民可以成为善良阵营的核心

### 1.4 在游戏流程中的作用

**游戏早期（1-2天）**：
- 洗衣妇是信息源头，需要决定何时、如何分享信息
- 可以通过私聊建立小圈子，验证其他玩家的发言
- 帮助善良阵营识别可信任的玩家

**游戏中期（3-4天）**：
- 洗衣妇的信息价值开始与其他信息源交叉验证
- 可能成为恶魔的击杀目标（如果公开身份）
- 需要平衡信息分享与自保

**游戏后期（5天以后）**：
- 首夜信息的价值可能降低（因为部分玩家已死亡）
- 但确认的镇民如果存活，可成为决胜关键
- 洗衣妇本人可能成为证明链条的一环

### 1.5 新手友好度分析

洗衣妇是最适合新手的角色之一，原因如下：

**优点**：
- **能力简单**：只在首夜获得信息，无需每晚行动
- **压力较小**：不像送葬者或占卜师需要每天做复杂判断
- **明确目标**：知道自己获得了什么信息，容易理解
- **学习机会**：通过观察被确认的镇民如何行动，学习游戏技巧

**需要注意**：
- 虽然能力简单，但**如何使用信息**是一门艺术
- 新手常犯错误：第一天就全盘托出，导致被恶魔针对
- 需要学习基本的社交推理和信息管理

## 2. 能力机制深度解析

### 2.1 触发时机与流程

洗衣妇的能力在游戏的 **"首个夜晚"（First Night）** 触发。具体流程如下：

**时间线**：
1. 所有玩家闭眼
2. 说书人按照夜晚顺序唤醒角色
3. 洗衣妇在首夜顺序中被唤醒（顺序编号：2）
4. 说书人指向两名玩家，然后展示一个镇民角色标记
5. 洗衣妇确认后闭眼
6. 游戏继续进行其他夜晚流程

**重要**：洗衣妇只在首夜行动一次，之后的每个夜晚都不会再被唤醒。

### 2.2 信息给予规则

说书人在给予信息时遵循以下规则：

#### 2.2.1 正常情况

当洗衣妇**健康且清醒**（没有醉酒也没有中毒）时：
- 说书人会指向两名玩家
- 其中**必有一人**是展示的那个镇民角色
- 另一人**不是**那个镇民角色（但可以是其他任何角色）

**示例场景**：
```
说书人展示：3号 + 7号，僧侣（Monk）

可能的真相：
- 3号是僧侣，7号是厨师（镇民）
- 3号是僧侣，7号是管家（外来者）
- 3号是间谍（爪牙），7号是僧侣
- 3号是小恶魔（恶魔），7号是僧侣
```

#### 2.2.2 醉酒状态（Drunk）

如果洗衣妇实际上是 **酒鬼（Drunk）**：
- 酒鬼会以为自己是洗衣妇
- 说书人会给予**虚假信息**
- 信息可能完全错误：两个玩家都不是展示的角色

**示例**：
```
你以为自己是洗衣妇，收到信息："5号和9号，其中一位是占卜师"
真相：你是酒鬼，5号是厨师，9号是间谍，根本没有占卜师在这两人中
```

**识别方法**：
- 如果你的信息与其他人的信息严重矛盾，可能是醉酒
- 如果确认的"镇民"行为怪异或信息错误，可能你是酒鬼

#### 2.2.3 中毒状态（Poisoned）

如果洗衣妇在首夜被 **投毒者（Poisoner）** 中毒：
- 投毒者在首夜的行动顺序（编号7）在洗衣妇（编号2）之后
- **正常情况下**，洗衣妇在首夜不会被毒
- 但在某些特殊机制下（如旅行者能力、酗酒者等），可能首夜就中毒

**中毒效果**：
- 收到的信息可能是假的
- 说书人可以选择给真信息或假信息

#### 2.2.4 间谍（Spy）特殊互动

**间谍**是一个特殊的爪牙，它"会在所有查验中被识别为善良、镇民，并被识别为与说书人选择的角色相同"。

这意味着：
- 说书人可以选择让间谍成为洗衣妇确认的"镇民"
- 即使间谍是邪恶的，洗衣妇的信息依然"正确"

**示例**：
```
说书人告诉你："2号和8号，其中一位是士兵"
真相：2号是间谍（被识别为士兵），8号是送葬者
你的信息技术上是"正确"的，但2号实际是邪恶阵营
```

**这是洗衣妇面临的最大陷阱**：即使信息"正确"，确认的"镇民"也可能是间谍。

#### 2.2.5 醉酒者（Drunk）与酒鬼的区别

很多新手会混淆这两个概念：

| 概念 | 含义 | 影响 |
|------|------|------|
| **酒鬼角色（Drunk）** | 一个外来者角色，以为自己是镇民但能力无效 | 如果你实际是酒鬼，你的所有信息都是假的 |
| **醉酒状态（Drunk/Poisoned）** | 能力被某种效果失效的状态 | 你是真洗衣妇，但首夜信息可能因中毒而错误 |
| **酗酒者（Sailor）** | 一个镇民角色，自己选择一人醉酒 | 首夜如果洗衣妇被酗酒者选中，会收到假信息 |

### 2.3 特殊情况处理

#### 2.3.1 被展示的镇民已死亡

在首夜时，所有玩家都是活着的，所以不存在"展示的镇民已死"的情况。但在某些自定义剧本或特殊规则下，如果有首夜死亡机制：
- 说书人不会展示已死的镇民
- 会选择其他存活的镇民

#### 2.3.2 两名玩家都是镇民

这是完全可能的：
```
说书人告诉你："4号和10号，其中一位是送葬者"
真相：4号是送葬者，10号是僧侣
```

这种情况下，你确认了一个送葬者，但也意外知道了10号"不是送葬者"。如果10号后续自称送葬者，你就知道他在撒谎。

#### 2.3.3 恶魔变身机制

在某些剧本中（非 Trouble Brewing），恶魔可能变身成其他角色。但在 Trouble Brewing 中：
- **小恶魔（Imp）** 不会变身成镇民
- 洗衣妇不会因恶魔变身而收到错误信息

#### 2.3.4 游戏中角色变化

如果在游戏进行中，你确认的镇民因某些机制变成了其他角色：
- **你首夜收到的信息依然是"正确"的**（因为首夜时他确实是那个镇民）
- 但他现在的身份可能已经不同

**示例**：
```
首夜：你确认5号是占卜师
第3天：5号因某种机制变成了酒鬼
结果：你的首夜信息没有错，但5号现在确实不是占卜师了
```

### 2.4 信息的确定性分析

洗衣妇的信息具有以下特点：

**高确定性**：
- 在健康状态下，信息100%准确
- 不像占卜师每晚可能收到新的错误信息
- 首夜信息是整个游戏的基石

**有限范围**：
- 只能确认"两个人中有一个是X"
- 无法确定具体是哪一个（需要通过发言和行为判断）
- 没有被展示的玩家完全是未知

**潜在陷阱**：
- 间谍机制可能让你确认"正确但无用"的信息
- 醉酒/中毒可能让信息完全错误
- 需要与其他信息源交叉验证

## 3. 玩法策略

### 3.1 信息利用的核心原则

洗衣妇的信息价值取决于**如何使用**，而不仅仅是获得了什么。以下是三个核心原则：

#### 原则1：信息是资产，不是负担
- 不要因为有信息就焦虑，觉得必须立刻分享
- 信息在你手中时，你掌握主动权
- 一旦公开，就无法收回

#### 原则2：时机决定价值
- 太早公开：成为恶魔首要击杀目标，信息未充分利用
- 太晚公开：错过建立信任网络的黄金期
- 最佳时机：通常是第1-2天，视场上局势而定

#### 原则3：部分公开胜过全盘托出
- 可以只公开部分信息
- 保留关键细节用于后续验证
- 通过私聊建立小圈子

### 3.2 发言时机策略

#### 3.2.1 第一天立刻公开（激进策略）

**适用场景**：
- 你确认的是关键强力角色（如僧侣、占卜师、送葬者）
- 想要快速建立善良核心
- 玩家群体较为和谐，恶魔不太会首夜杀信息角色

**操作方式**：
```
第一天发言：
"我是洗衣妇，首夜确认3号和7号中有一个僧侣。
我希望真正的僧侣能与我私聊，我们可以建立信任圈。"
```

**优点**：
- 快速建立善良核心
- 僧侣可以立刻知道有人确认他，增加配合
- 透明度高，获得其他善良玩家信任

**缺点**：
- 成为恶魔首要击杀目标
- 如果确认的镇民是间谍，你会被完全误导
- 失去后续利用信息的灵活性

#### 3.2.2 观望1-2天再公开（稳健策略）

**适用场景**：
- 想要先观察两名玩家的行为
- 场上有其他信息角色先跳出来
- 需要更多信息来判断谁是真镇民

**操作方式**：
```
第一天：保持沉默或模糊发言
第二天：观察3号和7号的发言和投票
第三天：根据判断，私聊你认为是真僧侣的人
```

**优点**：
- 有时间判断哪个是真镇民，哪个是干扰项
- 不会太早暴露身份
- 可以根据场上局势调整策略

**缺点**：
- 可能错过建立信任的黄金期
- 其他玩家可能因为你不跳身份而怀疑你
- 真镇民可能因得不到支持而被票出局

#### 3.2.3 全程隐藏（保守策略）

**适用场景**：
- 场上已有足够的信息角色跳出
- 你确认的镇民不是关键角色
- 想要作为"底牌"保存到最后

**操作方式**：
```
全程不公开身份，只通过私聊与确认的镇民建立联系
或者完全不主动使用信息，只在关键时刻（如最后几天）才揭示
```

**优点**：
- 不会被恶魔优先击杀
- 保留悬念，可以在关键时刻扭转局势
- 如果确认的镇民主动跳身份，你可以暗中支持

**缺点**：
- 信息价值大幅降低
- 可能被当作邪恶角色（因为不跳身份）
- 如果被提名处决，来不及利用信息

### 3.3 私聊策略

私聊是洗衣妇最重要的工具之一。

#### 3.3.1 何时发起私聊

**最佳时机**：
- 第一天白天讨论时间
- 在公开发言前先私聊确认的玩家
- 观察到某个玩家的发言符合你确认的角色

**私聊对象选择**：
1. **首选**：你认为是真镇民的那个人
2. **次选**：其他已跳身份的信息角色（建立信息网）
3. **备选**：看起来可信的其他玩家

#### 3.3.2 私聊话术示例

**与确认的镇民私聊**：
```
你："嗨，我是洗衣妇。首夜确认你和另一位玩家中有一个是僧侣。
     根据你的发言，我倾向于相信你是真僧侣。你能告诉我你的情况吗？"

对方（如果是真僧侣）："我确实是僧侣，首夜我保护了5号。很高兴有人确认我。"

你："太好了。我们可以互相确认身份。我建议我们先不要公开，
     等观察几天再决定。你有什么想法？"
```

**与其他信息角色私聊**：
```
你："我是洗衣妇，确认了一个僧侣。你是什么角色？我们可以交换信息。"

对方（如果是调查员）："我是调查员，首夜确认了一个男爵。我们的信息可以交叉验证。"

你："好的，让我们建立一个小圈子。我确认的僧侣是3号或7号之一..."
```

#### 3.3.3 私聊的风险管理

**风险**：
- 对方可能是邪恶角色，会把你的信息传给恶魔
- 私聊内容可能被曝光
- 过多私聊会被其他玩家怀疑

**应对**：
- 不要在私聊中透露全部信息，保留验证手段
- 观察对方是否会泄露你的秘密
- 适当在公开场合发言，保持存在感

### 3.4 身份隐藏技巧

有时候，不跳身份也是一种策略。

#### 3.4.1 伪装成其他角色

**可以伪装的角色**：
- **普通镇民**：没有特殊能力的角色（如处女、士兵）
- **未确认的角色**：假装你不知道首夜信息

**话术示例**：
```
公开场合："我是士兵，没有特别的信息提供。"
私下：与确认的镇民私聊，建立信任
```

**注意**：
- 如果伪装角色被真的那个角色拆穿，你会很被动
- 伪装时要符合角色逻辑（如士兵通常低调）

#### 3.4.2 模糊发言

不直接说自己是洗衣妇，但暗示有信息：
```
"我觉得3号玩家很可信，我有理由相信他是善良的。"
"根据我了解的情况，7号的发言有些可疑。"
```

这种方式既能影响局势，又不会完全暴露身份。

### 3.5 信息分享的层次设计

**三层信息分享模型**：

**第一层（核心圈）**：
- 你确认的镇民
- 其他可信的信息角色
- 分享：完整的首夜信息，包括具体玩家号码和角色

**第二层（扩大圈）**：
- 看起来可信的其他玩家
- 分享：部分信息，如"我确认了某个强力镇民"，但不说具体是谁

**第三层（公开场合）**：
- 所有玩家
- 分享：模糊信息或引导性发言，如"我认为某某号值得信任"

通过这种分层，你可以：
- 在不同圈子中验证信息
- 观察信息是否泄露，从而识别邪恶角色
- 保持灵活性

## 4. 角色互动与配合

### 4.1 与其他信息角色的协同

#### 4.1.1 与图书管理员（Librarian）配合

**协同价值**：
- 洗衣妇确认镇民，图书管理员确认外来者
- 两者信息不重叠，可以覆盖更多玩家
- 互相验证对方的身份

**配合策略**：
```
场景：你确认3号或7号是僧侣，图书管理员确认5号或9号是管家

策略：
1. 私下交换信息，建立核心信任圈
2. 如果3号既不是僧侣也不是管家，可以排除他
3. 共同观察确认的玩家，交叉验证
```

**案例**：
```
你的信息：2号和6号，其中一个是占卜师
图书管理员的信息：6号和10号，其中一个是管家

分析：
- 如果6号是管家，那么2号是占卜师
- 如果6号是占卜师，那么10号是管家
- 通过观察6号的发言，可以同时验证两条信息
```

#### 4.1.2 与调查员（Investigator）配合

**协同价值**：
- 洗衣妇确认善良镇民，调查员确认邪恶爪牙
- 信息互补，可以建立"白名单"和"黑名单"

**配合策略**：
```
场景：你确认4号或8号是僧侣，调查员确认5号或11号是男爵

策略：
1. 如果4号或8号跳出来承认是僧侣，可以初步信任
2. 如果5号或11号行为可疑，调查员的信息有助于确认
3. 两个信息角色互相背书，增加公信力
```

**注意陷阱**：
- 如果调查员确认的"爪牙"是间谍，他会伪装得很好
- 需要观察行为，不能只依赖信息

#### 4.1.3 与共情者（Empath）配合

**协同价值**：
- 共情者每晚知道邻座中有几个邪恶玩家
- 可以与洗衣妇的信息交叉验证

**配合策略**：
```
场景：你确认3号或5号是送葬者，共情者坐在2号和4号之间

共情者首夜得知：我的邻座中有1个邪恶玩家

分析：
- 如果3号是送葬者，那么5号可能是邪恶的
- 如果共情者的邻座（2号和4号）有一个邪恶，可以排除范围
```

#### 4.1.4 与占卜师（Fortune Teller）配合

**协同价值**：
- 占卜师每晚查验两名玩家，得知是否有恶魔
- 洗衣妇可以提供初始的善良玩家名单

**配合策略**：
```
场景：你确认6号或9号是僧侣

建议占卜师：
"我确认了6号或9号是僧侣，建议你优先查验这两个人，
 如果查到恶魔，说明我的信息有问题（醉酒/中毒）或者有间谍。"
```

### 4.2 与执行角色的配合

#### 4.2.1 与送葬者（Undertaker）配合

**协同价值**：
- 送葬者每晚知道白天被处决者的角色
- 可以验证被处决玩家是否是你确认的镇民

**配合场景**：
```
你的信息：7号或12号是占卜师

第2天：12号被投票处决
第2夜：送葬者得知12号确实是占卜师

结论：你的信息正确，7号不是占卜师
```

**反向利用**：
```
如果送葬者告诉你："被处决的人不是占卜师"
而你确认过那个人是占卜师
说明：你可能醉酒/中毒，或者送葬者在撒谎
```

#### 4.2.2 与僧侣（Monk）配合

**协同价值**：
- 如果你确认的就是僧侣，你们可以形成强力组合
- 僧侣可以保护你不被恶魔击杀

**配合策略**：
```
私聊僧侣："我确认你是僧侣，能否请你保护我？
            我们可以建立一个核心圈子。"

僧侣的选择：
- 同意保护你，让你可以安全地公开信息
- 或保护其他关键角色，你则保持低调
```

**高级玩法**：
```
如果恶魔知道你是洗衣妇且确认了僧侣，
他可能会避免击杀你（因为僧侣可能保护你）
反而去杀其他角色
这时你可以利用"诱饵"身份，保护真正的核心角色
```

#### 4.2.3 与士兵（Soldier）配合

**协同价值**：
- 士兵免疫恶魔击杀
- 如果你确认的是士兵，他可以大胆跳身份吸引火力

**配合策略**：
```
确认士兵后：
"我确认你是士兵，你可以主动跳出来吸引恶魔注意，
 保护其他关键角色。我会在暗中支持你。"
```

### 4.3 被邪恶角色针对时的应对

#### 4.3.1 应对小恶魔（Imp）

**恶魔的视角**：
- 洗衣妇是首夜信息角色，会优先考虑击杀
- 但如果洗衣妇被僧侣保护，击杀会失败

**你的应对**：
- **第一天不跳身份**：降低被杀风险
- **私下与僧侣联系**：请求保护
- **虚张声势**："我有关键信息，恶魔如果杀我会后悔"

**如果被杀**：
- 死亡前的最后发言很关键
- 尽量传递信息给可信玩家

#### 4.3.2 应对投毒者（Poisoner）

**投毒者的策略**：
- 通常会毒信息角色，让他们收到错误信息
- 但洗衣妇只在首夜行动，毒你的价值较低（除非首夜就毒）

**你的应对**：
- 如果怀疑自己首夜被毒（信息与他人矛盾），公开说明
- 与其他信息角色交叉验证

#### 4.3.3 应对间谍（Spy）

**最危险的敌人**：
- 间谍可以被识别为任何角色，包括你确认的镇民
- 你可能确认的"僧侣"其实是间谍

**识别方法**：
```
观察确认的"镇民"的行为：
- 是否过度积极地想知道其他人的信息？
- 是否在关键投票时偏向邪恶阵营？
- 他的信息是否与其他人矛盾？
```

**应对策略**：
- 不要100%信任确认的镇民
- 保留一些关键信息不告诉他
- 通过后续行为持续验证

#### 4.3.4 应对男爵（Baron）

**男爵的影响**：
- 男爵会增加外来者数量，减少镇民数量
- 意味着场上镇民变少，你确认的镇民更珍贵

**你的应对**：
- 更加珍惜你确认的镇民
- 与调查员配合，确认男爵位置
- 如果男爵跳身份，可以联合其他人投票处决

### 4.4 与外来者的互动

#### 4.4.1 与酒鬼（Drunk）的关系

**如果你是酒鬼**：
- 你以为自己是洗衣妇，但能力无效
- 收到的信息完全是假的
- 很难自我识别（除非信息严重矛盾）

**如果确认的"镇民"是酒鬼**：
- 酒鬼以为自己是某个镇民，但能力无效
- 说书人可能让你确认酒鬼，告诉你他是某个镇民
- 这时你的信息"技术上正确"（酒鬼以为自己是那个角色），但实际误导

#### 4.4.2 与管家（Butler）的互动

**管家的能力**：
- 管家必须跟随主人投票，否则当晚死亡

**如果你确认的是管家**：
- 管家是外来者，说书人不会给你这个信息（洗衣妇只确认镇民）
- 但你可以通过排除法推断：如果另一个人是镇民，那确认的另一人就不是镇民

## 5. 常见误区与陷阱

### 5.1 新手常犯错误

#### 5.1.1 第一天全盘托出

**错误示范**：
```
第一天发言："我是洗衣妇，首夜确认3号和7号中有一个是僧侣。
            僧侣请跳出来，我们组队吧！"
```

**问题所在**：
- 暴露了自己的身份（成为恶魔目标）
- 暴露了确认的镇民范围（恶魔会针对）
- 没有留下验证手段（无法判断后续谁在撒谎）
- 失去了信息的灵活性

**正确做法**：
- 私下与3号和7号分别接触
- 先观察他们的发言和行为
- 部分公开或延迟公开

#### 5.1.2 100%信任确认的镇民

**错误思维**：
"我确认了他是僧侣,所以他说什么我都信。"

**问题所在**：
- 可能是间谍（技术上也算"僧侣"）
- 可能是醉酒状态的酒鬼（以为自己是僧侣）
- 即使是真僧侣，也可能判断错误

**正确做法**：
- 持续观察行为
- 与其他信息交叉验证
- 保留怀疑态度

#### 5.1.3 忽略另一个玩家

**错误思维**：
"我确认3号和7号有一个是僧侣，我只关注这两个人就够了。"

**问题所在**：
- 另一个玩家也提供了信息（他不是僧侣）
- 如果那个玩家后续跳僧侣，你就知道他在撒谎
- 浪费了信息价值

**正确做法**：
```
记录信息：
- 3号或7号是僧侣
- 3号和7号中，必有一个不是僧侣
- 如果后续有第三个人跳僧侣，可以判断他是假的（除非你醉酒/中毒）
```

#### 5.1.4 死后不传递信息

**错误做法**：
你在第2夜被恶魔杀死，第3天白天你有最后发言机会，但你说：
"我没什么好说的，我死了。"

**问题所在**：
- 浪费了最后的发言机会
- 首夜信息没有传递给善良阵营
- 可能导致善良阵营失去关键线索

**正确做法**：
```
死亡发言："我是洗衣妇，首夜确认了3号或7号是僧侣。
           我生前倾向于相信7号是真僧侣，3号可疑。
           请大家关注这条信息，僧侣可以保护关键角色。"
```

### 5.2 中级玩家易犯错误

#### 5.2.1 过度依赖首夜信息

**错误思维**：
"我的信息是首夜就确定的，一定是最可靠的。"

**问题所在**：
- 游戏是动态的，情况会变化
- 可能醉酒/中毒
- 可能确认了间谍
- 其他玩家的信息可能更有价值

**正确做法**：
- 首夜信息是起点，不是终点
- 持续收集新信息
- 愿意调整判断

#### 5.2.2 忽视私聊反馈

**错误做法**：
```
你私聊3号："我确认你是僧侣。"
3号回复："啊？我不是僧侣啊，我是厨师。"
你想："他肯定在撒谎，我的信息不会错。"
```

**问题所在**：
- 没有考虑到你可能醉酒
- 没有考虑到另一个玩家（7号）才是真僧侣
- 固执己见会导致误判

**正确做法**：
- 如果对方否认，考虑多种可能性
- 去验证另一个玩家
- 观察对方后续行为

#### 5.2.3 信息泄露给间谍

**场景**：
```
你确认了5号是僧侣，于是与5号私聊分享了所有信息。
实际上5号是间谍，他把你的信息全部告诉了恶魔。
第二天晚上，恶魔根据你的信息杀死了关键角色。
```

**教训**：
- 私聊时保留部分信息
- 观察对方是否泄密
- 不要过早完全信任

### 5.3 高级玩家需注意的细节

#### 5.3.1 被邪恶角色反利用

**高级陷阱**：
```
场景：
- 你确认3号或7号是占卜师
- 3号是间谍，他知道真占卜师是7号
- 3号主动找你私聊："我是占卜师，感谢你确认我。"
- 你信任了3号，忽略了7号
- 7号（真占卜师）因为得不到支持而被投票出局
```

**应对**：
- 即使对方主动承认身份，也要验证
- 观察两个人的后续行为
- 不要过早站队

#### 5.3.2 信息武器化

**陷阱**：
邪恶阵营可能故意引导你错误使用信息：
```
场景：
- 你确认4号或9号是送葬者
- 恶魔阵营故意让4号跳送葬者（4号是爪牙）
- 你公开支持4号，增加他的可信度
- 9号（真送葬者）被当作假跳，被投票出局
```

**应对**：
- 不要公开背书太早
- 让确认的玩家通过行为证明自己
- 保持怀疑态度

#### 5.3.3 忽视"负面信息"

**常见疏忽**：
```
你的信息：2号和8号，其中一个是占卜师

后续发现：
- 6号跳占卜师
- 你的第一反应："6号是假跳！"

但你忽略了：
- 你可能醉酒/中毒
- 可能有多个占卜师（某些机制）
- 6号可能是真的，2号和8号都不是占卜师
```

**正确思维**：
- 信息给了你"可能范围"，不是"绝对真相"
- 要验证而不是断言

### 5.4 说书人常见失误（玩家需了解）

虽然你是玩家不是说书人，但了解说书人可能的失误可以帮助你判断：

#### 5.4.1 给错信息

说书人可能不小心：
- 展示了错误的角色标记
- 指向了错误的玩家

**你的应对**：
- 如果信息严重不合逻辑，可以礼貌地私下询问说书人
- 但不要假设说书人出错，优先考虑游戏内的原因（醉酒/中毒）

#### 5.4.2 忘记间谍机制

有些新手说书人可能忘记：
- 间谍可以被识别为任何角色
- 给洗衣妇信息时，可以让间谍成为"确认的镇民"

**影响**：
- 如果说书人不熟悉规则，你确认的镇民更可能是真的
- 但不要依赖这一点

## 6. 高级技巧

### 6.1 信息分层与验证网络

#### 6.1.1 建立三重验证体系

**第一重：直接验证**
```
你确认3号或7号是僧侣
→ 私聊3号和7号，观察谁的反应更像真僧侣
→ 真僧侣通常会：
  - 询问你的完整信息
  - 分享自己首夜保护了谁
  - 表现出合作意愿
```

**第二重：行为验证**
```
观察两人的后续行为：
- 真僧侣每晚保护人，可能会暗示"某某人很安全"
- 假僧侣（邪恶角色）可能会：
  - 避免具体细节
  - 行为与僧侣逻辑不符
  - 在关键时刻暴露
```

**第三重：信息交叉验证**
```
与其他信息角色合作：
- 占卜师查验你确认的两人
- 共情者观察他们的邻座
- 送葬者如果他们被处决，能确认真身
```

#### 6.1.2 "诱饵信息"技术

**高级玩法**：给不同的人不同版本的信息，看哪个泄露

```
操作：
- 私聊玩家A："我确认3号是僧侣"
- 私聊玩家B："我确认7号是僧侣"
- 观察恶魔的击杀目标

分析：
- 如果恶魔杀了3号，说明A可能是邪恶的（泄露了信息）
- 如果恶魔杀了7号，说明B可能是邪恶的
```

**风险**：
- 如果被发现你给了不同信息，会失去信任
- 只在高水平局中使用

### 6.2 心理战与博弈

#### 6.2.1 "假醉酒"战术

**场景**：你收到了清晰的信息，但故意表现得像醉酒

```
公开发言："我是洗衣妇，但我的信息很奇怪，
           可能我醉酒了。我确认的人行为完全不像镇民。"

实际：
- 你的信息是正确的
- 你故意这样说，降低恶魔对你的重视
- 暗中与确认的镇民建立信任
```

**效果**：
- 恶魔可能不会优先杀你（觉得你信息无用）
- 你可以更安全地行动

#### 6.2.2 "反向钓鱼"

**场景**：故意公开错误信息，引诱邪恶角色上钩

```
公开发言："我确认了5号是占卜师。"
实际：你确认的是5号或9号，你倾向于相信9号是真的

邪恶角色可能：
- 5号如果是邪恶的，会顺势跳占卜师
- 真占卜师9号会跳出来反驳
- 通过反应可以判断谁是真的
```

**风险**：
- 善良阵营也会被误导
- 可能失去信任

#### 6.2.3 "沉默权威"

**技巧**：不跳身份，但通过行为建立权威

```
操作：
- 全程不说自己是洗衣妇
- 但在投票和发言中，始终支持你确认的镇民
- 当有人质疑他时，你坚定支持

效果：
- 其他玩家会好奇你为什么这么信任某人
- 真镇民可能察觉到你知道他的身份
- 私下建立信任，公开保持神秘
```

### 6.3 多层博弈策略

#### 6.3.1 第一层：信息本身

**基础层**：你确认了3号或7号是僧侣
- 这是你知道的事实
- 决定相信哪一个

#### 6.3.2 第二层：信息的使用

**策略层**：如何利用这个信息
- 公开还是私下？
- 何时分享？
- 分享给谁？

#### 6.3.3 第三层：预判他人的预判

**博弈层**：假设邪恶阵营知道你是洗衣妇
```
邪恶阵营会想：
"洗衣妇可能确认了某个强力镇民，我们要找出来杀掉。"

你的应对：
- 故意表现得对某个无关紧要的玩家很信任
- 引导恶魔杀错人
- 真正的确认对象保持低调
```

#### 6.3.4 第四层：利用信息不对称

**高级博弈**：
```
场景：
- 你确认了4号或11号是占卜师
- 你发现4号是真占卜师
- 但你公开说："我倾向于相信11号"

目的：
- 让恶魔以为11号是占卜师，去杀他
- 11号不是占卜师，恶魔浪费了击杀机会
- 真占卜师4号多活一天，能提供更多信息
```

### 6.4 极限操作

#### 6.4.1 "自爆换信任"

**场景**：游戏后期，需要证明自己

```
操作：
"我一直没跳身份，现在我必须告诉大家：
 我是洗衣妇,首夜确认了X号是某某角色。
 我知道自爆会让恶魔今晚杀我，但我愿意用命换取大家对X号的信任。
 请保护X号，他是关键。"

效果：
- 极大增强了X号的可信度
- 即使你被杀，信息传递成功
- 善良阵营获得明确的信任锚点
```

#### 6.4.2 "假跳恶魔"

**极端战术**：故意跳恶魔，引起混乱

```
场景：你判断善良阵营要输了，需要搏一把

操作：
"其实我是小恶魔，我认输了。请投票处决我。"

可能结果：
- 真恶魔可能慌乱，暴露破绽
- 善良阵营重新审视局势
- 你被处决后，送葬者确认你不是恶魔，反而证明了你的善良
```

**风险极高**：
- 可能被立刻处决，浪费投票机会
- 善良阵营可能不信任你
- 只在绝境时使用

#### 6.4.3 "信息遗嘱"

**技巧**：提前准备死亡发言

```
在笔记本上写：
"如果我死了，请告诉大家：
 - 我是洗衣妇
 - 确认了X号或Y号是某某角色
 - 我倾向于相信X号
 - 请重点保护X号
 - 我的其他观察：..."
```

**作用**：
- 即使突然死亡，也能传递完整信息
- 显得更专业，增加可信度

## 7. 说书人指引

虽然你是玩家，但理解说书人的职责可以帮助你更好地玩洗衣妇。

### 7.1 如何给洗衣妇信息

#### 7.1.1 基本原则

**健康洗衣妇**：
- 必须给真实信息
- 选择两名玩家，其中一人是指定的镇民

**醉酒/中毒洗衣妇**：
- 可以给假信息
- 两人都不是指定的镇民
- 或者根本没有那个镇民在场

#### 7.1.2 选择玩家的策略

**说书人应考虑**：

**平衡性**：
- 不要总是给最强镇民（如占卑师、僧侣）
- 适当给一些次要镇民（如厨师、图书管理员）
- 让游戏有多样性

**剧情性**：
- 考虑两个玩家的座位关系（邻座？对面？）
- 考虑他们的经验水平（新手vs老手）

**间谍利用**：
- 如果场上有间谍，可以让间谍成为"确认的镇民"
- 增加游戏复杂度

**示例**：
```
场景：13人局，有间谍

说书人的选择：
- 选项A：展示2号（真僧侣）和5号（厨师）
  → 洗衣妇容易判断，游戏较简单

- 选项B：展示3号（间谍，识别为占卜师）和7号（送葬者）
  → 洗衣妇可能被误导，游戏更复杂

- 选项C：展示9号（真占卜师）和11号（小恶魔）
  → 极端对比，容易判断
```

#### 7.1.3 特殊情况处理

**如果没有足够的镇民**：
- 标准情况下不会发生
- 但在自定义剧本中，说书人需要确保有镇民可以展示

**如果洗衣妇本身是酒鬼**：
- 展示任意两名玩家和任意镇民角色
- 可以完全随意，增加混乱

### 7.2 平衡性考量

#### 7.2.1 避免"确认环"

**什么是确认环**：
```
洗衣妇确认A是僧侣
图书管理员确认B是管家
调查员确认C是间谍
占卜师查验D不是恶魔
...

结果：善良阵营确认了太多人,游戏失去悬念
```

**说书人应对**：
- 适当给洗衣妇不太强的镇民
- 或者让洗衣妇确认间谍
- 保持信息的模糊性

#### 7.2.2 信息的"误导价值"

**好的信息设计**：
```
给洗衣妇：2号和8号，其中一个是占卜师

设计：
- 2号是真占卜师，但性格内向，不太发言
- 8号是间谍，表现得很像占卜师

效果：洗衣妇需要通过观察才能判断，增加游戏深度
```

### 7.3 特殊Ruling（规则裁定）

#### 7.3.1 角色变化情况

**问题**：如果首夜后，确认的镇民变成了其他角色怎么办？

**裁定**：
- 洗衣妇的信息在首夜时是正确的
- 后续变化不影响首夜信息的"正确性"

**示例**：
```
首夜：洗衣妇确认3号是占卜师（当时确实是）
第2天：3号因某种机制变成了酒鬼
结果：洗衣妇的信息"没有错"，只是3号现在不是占卜师了
```

#### 7.3.2 多个同角色情况

**问题**：如果场上有多个同一镇民角色（某些自定义剧本）？

**裁定**：
- 说书人可以选择展示其中任意一个
- 洗衣妇的信息依然有效

#### 7.3.3 洗衣妇死亡后

**问题**：洗衣妇死后，她的信息还有效吗？

**裁定**：
- 首夜获得的信息永久有效（除非当时醉酒/中毒）
- 洗衣妇死亡不影响信息的真实性

### 7.4 说书人常见错误

#### 7.4.1 总是给最强镇民

**错误做法**：
每次都给洗衣妇确认占卜师、僧侣、送葬者等关键角色

**问题**：
- 游戏变得模式化
- 洗衣妇的价值过高，必然被恶魔优先杀

**正确做法**：
- 混合给强力和次要镇民
- 让每局游戏都有新鲜感

#### 7.4.2 忘记考虑间谍

**错误做法**：
有间谍在场，但说书人从不让间谍成为"确认的镇民"

**问题**：
- 浪费了间谍的机制
- 游戏缺少深度

**正确做法**：
- 适当让间谍被确认
- 增加游戏的不确定性

#### 7.4.3 信息过于明显

**错误做法**：
```
给洗衣妇：1号（新手，很害羞）和6号（老手，很活跃），其中一个是占卜师

结果：明显6号更像占卜师，洗衣妇很容易判断
```

**正确做法**：
- 选择两个难以区分的玩家
- 或选择性格类似的玩家

分析：
- 9号很热情
- 主动提出建立信任圈
- 看起来很合理

**第1天投票**：

**第1夜**：
- 你没有夜晚能力，等待结果

问题出现了：

- 9号可能太急于"表现"

**第2天深夜私聊**：
你找到真正的图书管理员（4号）：

4号："我是图书管理员，首夜确认了10号或12号有一个是管家。

分析：
- 4号是真图书管理员
- 3号撒谎了
- 那么3号很可能是真占卜师，只是隐藏身份

"2号确实是士兵。"

```
9号的查验错了！
可能性：
1. 9号是酒鬼（以为自己是占卜师）
2. 9号被投毒了
3. 9号是邪恶角色假跳占卜师
4. 9号是真占卜师，但当时醉酒/中毒

```
重新审视首夜信息：
- 3号和9号，其中一个是占卜师

- 那么3号更可能是真占卜师

但3号说自己是图书管理员...
等等，如果3号是间谍呢？

间谍可以被识别为任何角色，包括占卜师！
我的信息"技术上正确"，但3号可能是间谍！

**第3天白天**：

**案例总结**：

2. **需要交叉验证**：不能只依赖自己的信息，要与其他人配合
3. **行为比身份更重要**：9号虽然跳了占卜师，但行为暴露了问题

## 总结

Washerwoman（洗衣妇）是血染钟楼中一个看似简单但实则深邃的角色。首夜的信息是你的起点，但如何使用这个信息，才是决定胜负的关键。

**核心要点回顾**：

1. **信息管理**：不要第一天就全盘托出，学会分层分享
2. **交叉验证**：你的信息不是绝对真理，需要与其他角色配合
3. **警惕陷阱**：间谍、醉酒、中毒都可能让你的信息误导你
4. **行为观察**：确认的"镇民"需要通过行为持续验证
5. **灵活应变**：根据局势调整策略，不要固守首夜信息
6. **团队合作**：建立核心圈子，与其他信息角色协同
7. **诚实价值**：如果发现自己是酒鬼，诚实承认反而能赢得信任

**最后的建议**：

血染钟楼是一个社交推理游戏，技术和策略固然重要，但**人与人之间的信任、沟通和博弈**才是游戏的灵魂。作为洗衣妇，你拥有了第一手信息，这既是优势也是考验。

愿你在每一局游戏中，都能善用你的洗衣盆，洗净真相，揭露谎言！
